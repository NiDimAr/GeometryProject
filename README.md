# GeometryProject

Многомодульный Maven-проект для работы с геометрическими фигурами.

## Структура проекта

GeometryLibrary — основная библиотека с геометрическими фигурами  
GeometryApp — приложение, использующее библиотеку  
GeometryUtils — модуль с утилитами для работы с геометрией  
Geometry3D — модуль с трёхмерными геометрическими фигурами  

## Возможности

### 2D фигуры
- Круг
- Прямоугольник
- Треугольник

Каждая фигура поддерживает:
- вычисление площади
- вычисление периметра

### Утилиты
GeometryUtils предоставляет:
- сравнение фигур по площади
- сравнение фигур по периметру
- дополнительные вспомогательные методы для работы с геометрическими данными

### 3D фигуры
- Куб
- Сфера

Поддерживаются базовые вычисления объёма и площади поверхности (в зависимости от реализации)

## Технологии
- Java 17
- Maven (многомодульный проект)
- Объектно-ориентированное программирование (ООП)
- Git (контроль версий)
